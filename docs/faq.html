Installazione e primo avvio

AnimeUnion Docker gira come container sul tuo NAS o server. Con Docker Compose bastano pochi minuti: 

docker compose up -d

Le cartelle. Il container vede un percorso interno (di solito /media): nel docker-compose.yml ci mappi la tua libreria reale (es. /volume1/Anime:/media/Anime). Nell'app, alla voce Cartelle di download, scegli quel percorso interno con Sfoglia. Se una cartella risulta «non scrivibile / non montata», controlla il volume nel compose.

Il wizard. Al primo accesso una procedura guidata ti fa impostare login, cartelle (con verifica di scrivibilità), aspetto e, se vuoi, Jellyfin. Puoi rivedere e cambiare tutto in seguito da Impostazioni.

HTTPS e Tailscale

Alcune funzioni del browser — notifiche push e installazione come app (PWA) — funzionano solo su un indirizzo sicuro HTTPS (oppure su localhost). È una regola del browser, non dell'app: se apri AnimeUnion via http://192.168… quelle funzioni restano disattivate.

La via più semplice è Tailscale. Ti dà un indirizzo HTTPS valido in automatico, senza dominio né porte aperte sul router:

  1. Installa Tailscale sull'host/NAS e fai il login (stessa tailnet del telefono/PC).
  2. Nella console Tailscale abilita HTTPS Certificates per la tua tailnet (una volta sola).
  3. Esponi l'app con tailscale serve (o l'integrazione del tuo NAS) e apri l'indirizzo https://<nome>.<tailnet>.ts.net dal telefono.

Dettagli passo-passo (anche per reverse proxy con dominio proprio) nel README, sezione HTTPS.

App installabile e notifiche push

Puoi installare AnimeUnion come app dedicata (PWA) sul telefono o sul desktop, e ricevere notifiche push del browser (episodio completato, nuove stagioni) anche ad app chiusa.

Requisito: serve HTTPS (vedi sopra). Su un indirizzo non sicuro il pulsante «Installa» e il toggle delle push non compaiono — non è un bug.

  • Installare: apri l'app in HTTPS, poi usa «Installa» dal menu del browser (o il pulsante in Impostazioni › Notifiche quando disponibile).
  • Push: in Impostazioni › Notifiche attiva «Push del browser» e concedi il permesso quando richiesto.

Download neurale (worker GPU)

Il download neurale migliora la qualità di un episodio già scaricato portandolo a XQ (1080p) o XQ+ (4K) con gli shader Anime4K (gli stessi del player del sito). La sorgente SD resta sempre disponibile: l'upscale crea un file in più, non sostituisce l'originale. È una funzione Premium.

Perché serve un PC con GPU. L'upscale è pesante e richiede una scheda video con Vulkan; il NAS in genere non ce l'ha. Per questo il render gira su un worker separato (un tuo PC con GPU nella stessa rete), e l'app gli manda i lavori via LAN.

Come si configura:

  1. Sul PC con la GPU avvia il worker (vedi apps/worker/README.md): serve ffmpeg con libplacebo + Vulkan.
  2. Il worker espone un URL (es. http://192.168.1.20:8787) e usa un token condiviso.
  3. In Impostazioni › Download Neurale incolla URL e token, salva e premi «Verifica worker».
  4. Quando lo stato è «Pronto», dal catalogo scegli «Migliora a XQ/XQ+» su un episodio già scaricato.

Se lo stato resta «Non pronto»: controlla che il PC sia acceso e il worker avviato, che URL e token combacino, e che ffmpeg sul worker abbia libplacebo + Vulkan.

Jellyfin

I file vengono rinominati con il layout che Jellyfin riconosce (<Titolo>/Season 02/<Titolo> - S02E01.mp4) e organizzati per serie e stagione, con sidecar .nfo.

In Impostazioni › Jellyfin puoi inserire URL del server e API key e attivare il refresh automatico: dopo ogni nuovo episodio l'app chiede a Jellyfin di aggiornare la libreria interessata, così l'episodio compare senza scansioni manuali.

Backup del database

L'app fa un backup automatico del database SQLite (follow, stato dei download, impostazioni). I file video non sono nel backup: sono già sul tuo disco.

Se vuoi una copia fuori sede, in Impostazioni › Backup puoi collegare Google Drive (client OAuth «Desktop», scope minimo drive.file): l'app carica il backup più recente e mantiene solo gli ultimi secondo la retention scelta.

Upscale di anime già scaricati

Il download neurale è disponibile sugli episodi scaricati dall'app: li trovi nel catalogo con la voce «Migliora a XQ/XQ+» quando il worker è pronto. L'originale SD non viene toccato — l'upscale è un file aggiuntivo nella tua libreria.

L'apertura anche agli anime collegati senza scaricare è in valutazione: verrà abilitata dopo un collaudo dedicato del worker su file di provenienza arbitraria.
